Part I: What is Linux? Why use it?

Linux is an open source operating system (OS) that was developed as a free alternative to Unix. Linux was created by Linus Torvalds, re-using much software from the GNU (GNU's Not Unix) project which had similar goals. Linux is open source, which means that the actual code used to program the operating system can be viewed by anyone. This allows anyone (with enough programming skills) to check for security holes, add new features, and make the operating system more efficient and reliable. We recommend using Linux because it is free, open source, and often not much different than Windows or OS X.

Free and Free

Almost all versions of Linux are free (as in no cost). Most versions of Linux are also Free Software, meaning free as in "Free Speech" not free as in "Free Beer". Free Software means that copyrights or licensing must allow anyone to use the software for any purpose, change the software, and redistribute the original software or modified version. [More info]

Open Source

As a prerequisite to being Free Software, the code must be open source. Otherwise it would be difficult to study or change. Opting for open source software is like supporting transparency in government. In order to feel comfortable allowing the program to control aspects of your computer (or for the government to control aspects of your life), it is important to know what it's actually doing. Open source software allows anyone to look at the code for backdoors, vulnerabilities, inefficiencies, design problems, or even typographical errors. Closed source applications can only claim that they are following the rules or protecting your privacy, open source applications can prove it.

Easy to Use

Now that we have the more philosophical stuff out of the way, it's time for the good news: Linux is almost as easy to use as Windows or OS X. A decade ago, you had to be very computer savvy in order to even bother attempting to use Linux, but that has changed as Linux has grown in popularity. We picked Linux Mint because it supports full disk encryption out of the box, and has a lot of useful pre-installed software. Linux Mint is especially easy to use for Windows users, since programs are listed in a menu much like the Start Menu.

Part II: Installing Linux Mint

Download and Burn to DVD

  1. First, go to the Linux Mint download page.
  2. Select either the 32-bit or 64-bit download link next to Cinnamon. If you aren't sure which one, start with the 64-bit version and try the 32-bit version if that does not work. Most new computers will support the 64-bit version.
  3. On the next page, select a link from under the MIRROR column, or use the Torrent link near the top of the page.
  4. Once you have saved the .iso file to your computer, burn it to a DVD using a DVD-burning program or one of the methods described here.

Boot Your Computer from the DVD

  1. While the computer is still turned on, insert the DVD in to the drive.
  2. Shut down the computer completely.
  3. If you have an Apple computer, turn the computer on and hold the option key from the time you hear the sound to the point where you see an image of a hard drive and a CD on the screen. Use the arrow keys to select the CD and press enter.
  4. If you have a PC, turn the computer on and immediately start tapping the F12 key to get to the boot device menu. Select the CD/DVD drive using the arrow keys, and press enter. If this does not work for you, see this guide.

Install Linux Mint

  1. Your computer should start to the Linux Mint Live Desktop. This will work just like Linux Mint once it is installed, so here is your chance to play around before you commit to making the switch. No files on your computer will be changed, so you can feel free to explore.
  2. When you are ready to install, double-click the Install Linux Mint icon on the Desktop.
  3. Select the language you want to use with Linux Mint, then click Continue.
  4. Ensure that your computer meets the requirements, then click Continue.
  5. For the purposes of this guide, we're going to use Erase disk and install Linux Mint option. If you're feeling adventurous, you can use the other option, but that is far beyond the scope of this guide.
  6. We highly recommend that you select Encrypt the new Linux Mint installation for security option. This will encrypt your entire hard drive, so the computer cannot be started without the passphrase.
  7. We also recommend that you select the Use LVM with the new Linux Mint installation option. This will be helpful if you ever decide to make new partitions on your hard drive.
  8. Choose a strong password that will be used every time you start your computer. Type it again to confirm it.
  9. We highly recommend that you select the Overwrite empty disk space. This will write random data over the empty parts of the hard drive, so it will be virtually impossible for an adversary to determine where your encrypted files end and the empty space begins. With this option selected, the installation will take much longer, but we are of the opinion that this is time well spent. Make some margaritas to help pass the time.
  10. Select a location. This is mostly used for your time zone, but you may want to choose a different location in the same time zone (or even another time zone) for privacy reasons.
  11. Select your keyboard layout (you probably don't need to change it). You can test the setting by typing in the text box provided. If the right characters are appearing, you have the right keyboard setting.
  12. Enter your name (or any name...) and a name for your computer, then choose a username to log in with, and a password for your user account. Please note that this password can and should be different than the password you selected to encrypt your whole computer in a previous step.
  13. We also recommend that you select the options Require my password to log in and Encrypt my home folder for added security.
  14. Sit back and relax while the installation completes. Take a sip of your margarita and click through the slides to learn more about the software and features included in Linux Mint.
  15. When you see the Installation Complete box, you're nearly done! Remove the DVD from the drive, and click Restart Now to boot up your new Linux Mint installation.

Part III: Start and Update Linux Mint

  1. If you selected the option to encrypt the whole computer, you'll be presented with this screen. Type the computer password and press the enter key to decrypt and boot your computer.
  2. After you have entered your password to decrypt the computer, you will reach the main login screen. Click on your username.
  3. Enter the password for your user account and press the enter key or click OK.
  4. Once the Linux Mint desktop has loaded, we'll want to update the software. Click on the shield icon near the clock in the bottom right of the screen. Enter your user account password to launch the Update Manager. Click on Refresh to update the list, then Install Updates to install the updates.
  5. Once again, sit back and relax while the updates are installed. This could take some time.
