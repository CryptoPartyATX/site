Text Message (SMS) Encryption with TextSecure

A guide to encrypting text messages on Android phones

Purpose:This guide attempts to explain how to install and use the Android application TextSecure to send and receive encrypted text messages (SMS). TextSecure provides two main advantages over the built-in Android text message app. First, it encrypts the messages on your device, so they are private if your phone is lost or stolen. Second, it allows you to send encrypted text messages to other people who have the TextSecure software. Apart from these benefits, TextSecure also handles normal unencrypted text messages well; I personally prefer it to the built-in Android app even without the encryption.

Please note that when using TextSecure, only the body of the message is encrypted. Your phone carrier or anyone with access to their data can still see the metadata. That is, they can tell that a message was sent, to whom and from whom, when, etc. They will not be able to read the message body if it is encrypted. (Normally your carrier can read the body of your texts.)

Software: The only software needed for this guide is TextSecure. It is available on the Google Play Store. You will need an Android phone. The developer, Open WhisperSystems, is currently working on an iOS version, and it should be available soon.

Nutshell

First, install TextSecure from the Google Play Store.

Second, set a passphrase and (optional) import your current text messages.

Third, send an encrypted message to test the app.

Part I: Install TextSecure

  1. Search for TextSecure in the Google Play Store app on your phone. Click the INSTALL button.
  2. Review the App permissions screen. Since you will be using this software to handle your text messages, it will need to have access to them. If you agree to the permissions, press ACCEPT.
  3. The app will now download and install on your phone. When the installation is finished, you will see two buttons, OPEN and UNINSTALL, where the INSTALL button was previously. Click the OPEN button to launch the app.

Part II: Configure TextSecure

  1. The first time you run TextSecure, it will ask you to create a passphrase. Choose a passphrase that is as long and complex as you can possibly tolerate typing on a regular basis. You will need to type this passphrase every time you need to unlock TextSecure to read your messages. You can change this passphrase at any time, and you will be able to set how often you have to enter it. Type your passphrase and click Continue.
  2. On the Import screen, you have the option to copy all of your current text messages to the encrypted database. I recommend doing this now, so you can start using TextSecure as your text messaging program just like normal. You can always do it later if you want, and importing it does not delete the original. This also means that your original text messages are still stored unencrypted in your normal text message app. You may want to delete the originals after you have verified that the encrypted copies work. All future messages will be delivered to the TextSecure program and will be stored encrypted, whether or not the conversation was encrypted.
  3. That's all it takes. Now it's time to test.

Part III: Test TextSecure

  1. The hard part: find someone else who has TextSecure. The application is currently only available on Android, and in order to have an encrypted conversation both parties will need to have TextSecure installed. Please note that you will always be able to send unencrypted messages to anyone, just like normal. So find someone else and get them to install TextSecure. Then continue.
  2. Press the Add Message icon at the bottom left of the screen, or open the conversation you already have with the person you want to text. If it is a new conversation, type the phone number you want to contact in the top box.
  3. Before sending a message, click the lock at the top of the screen, and select Start Secure Session. A box will pop up asking you to confirm. Click Yes.
  4. TextSecure will send a key exchange message to the recipient. If the recipient is using TextSecure, you will receive a key exchange message in return. At that point, the arrow next to the “Type message” field will show a lock, and the conversation will now be encrypted.
  5. After that is complete, you can send and receive texts just like normal, and as long as the lock is still there on the send arrow, the contents will be encrypted!
